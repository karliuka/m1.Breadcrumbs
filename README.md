# Magento Breadcrumbs
Add breadcrumbs to any page.

<img alt="Magento Breadcrumbs" src="https://karliuka.github.io/m1/breadcrumbs/login.png" style="width:100%"/>

Download the plugin here: https://github.com/karliuka/m1.Breadcrumbs/archive/1.1.4.zip
Installation
-------

Download the plugin and copy the 'app' and 'skin' folder to the root of your Magento installation.
